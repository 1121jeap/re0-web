# 登场人物

------

## 爱蜜莉雅阵营

| 角色介绍 | 人物形象 |
|:---|:---:|
| **菜月昴（ナツキ・スバル，[Subaru Natsuki]）**<br/>主人公，17岁→18岁，高中三年级，经常逃课，被当作“惧学症学生”。<br/>不明原因被召唤到异世界，会使用阴属性魔法，并拥有“死亡回归”的特殊能力，<br/>即在异世界死亡后能回到悲惨事情发生之前，<br/>但每次回归都会增加“魔女的香味”的浓度，且无法对他人说出真相。<br/>在不断的死亡轮回中，爱上了爱蜜莉雅和雷姆，为了保护所爱之人竭尽全力。 | <img width="150" src="/res/img/character/486.jpg"> |
| **爱蜜莉雅（エミリア，[Emilia]）**<br/>女主角，也译作艾米莉娅。<br/>梦幻的银发，如雪的肌肤，魅惑的蓝紫色瞳孔，以及如精灵的尖耳。<br/>半妖精的美少女，也是精灵术士，因为外貌与传说中的嫉妒魔女·莎缇拉相像而常被人忌讳。<br/>曾被称为活在结冻之森的“冰结魔女”，初登场年龄114岁，心理年龄只有14岁。<br/>（7岁冰冻，沉睡100年后被帕克救出，生活7年后遇到昴。）<br/>老好人的善良性格，但本人坚决否认。因为很少与人打交道，所以只会在猫精灵帕克面前露出撒娇般的表情。<br/>露格尼卡王国的5位王位继承者候选人之一，常被菜月昴爱称为EMT。 | <img width="150" src="/res/img/character/Emilia.jpg" /> |
| **帕克（パック，[Pack]）**<br/>灰色的体毛，大大的眼睛，粉红色的鼻子，手掌大小的身体，双脚行走的小猫。<br/>早九晚五生活的男性精灵，时刻伴随在爱蜜莉雅的身边，爱称她为莉雅，是父亲一般的存在。<br/>真身是持有金色光辉的巨兽，长期居住在艾利欧鲁大森林永久冻土世界、<br/>掌管火之玛那，被称为“永久冻土”的终焉之兽的古老大精灵。<br/>无法忍受失去爱蜜莉雅的世界，立下契约，在爱蜜莉雅死后进行毁灭世界的行动。 | <img width="150" src="/res/img/character/Pack.jpg" /> |
| **雷姆（レム，[Rem]）**<br/>鬼族，罗兹瓦尔宅邸工作的双胞胎女仆的妹妹，特征是蓝发与右刘海。<br/>虽然外貌与姐姐几乎一样，但是家务万能，胸部也比较大。假献殷勤的毒舌担当，眼神温柔稳重，<br/>虽然一开始对昴抱有极强的敌意甚至下了杀手，但是最终爱上了昴。 | <img width="150" src="/res/img/character/Rem.jpg" /> |
| **拉姆（ラム，[Ram]）**<br/>鬼族，罗兹瓦尔宅邸工作的双胞胎女仆的姐姐，特征是红发与左刘海。<br/>不擅长料理，对罗兹瓦尔绝对忠诚与爱慕，拥有操纵风的能力。<br/>曾是鬼族中最强大的存在，号称“无角的鬼神”。<br/>在魔女教袭击村落时，被砍掉了力量的源泉“角”后弱化，依靠罗兹瓦尔的魔力补给。<br/>桀骜不驯的毒舌担当，眼神、言行都很紧，负责教育作为男仆的昴，并故意称呼他为巴鲁斯（卯日）。 | <img width="150" src="/res/img/character/Ram.jpg" /> |
| **碧翠丝（ベアトリス，[Beatrice]）**<br/>大精灵，外表是十一二岁的可爱萝莉，金色的长螺旋状卷发，穿着华丽的礼服。<br/>罗兹瓦尔宅邸禁书库的图书管理员，自称贝蒂。很粘帕克，称帕克为哥哥。<br/>虽然很不直率而且毒舌，但本性善良。曾为了帮助昴而与他缔结契约。 | <img width="150" src="/res/img/character/Beatrice.jpg" /> |
| **罗兹瓦尔·L·梅札斯（ロズワール・L・メイザース，[Roswaal L Mathers]）**<br/>左黄右蓝的不同瞳色，深蓝色的披肩长发，惨白的肤色与纤细的身形，画着小丑般的脸妆以及配色奇特的服饰。<br/>宅邸的主人，爱蜜莉雅候补人的后援支持者。卢克尼卡王国的边境伯爵，也绰号“宫廷魔法师”，<br/>是王国最厉害的魔法师，收留拉姆姐妹，并让她们在宅邸工作。 <br/>愿望是弑龙，为此聚集可利用的棋子。<br/> | <img width="150" src="/res/img/character/Roswaal.jpg" /> |
| **奥托·思文（オットー・スーウェン）**<br/>五官端正，头脑机智，还有得天独厚的加护，但却是被幸运抛弃的青年。<br/>作为一名行商人，有作为商人的头脑，但却没有受到幸运青睐。<br/>一头灰发，身高比昴略矮，脸颊瘦长。<br/>靠着走商讨生活，持有 “言灵的加护”，能与动物进行交流。 | <img width="150" src="/res/img/character/Otto.jpg" /> |
| **佩特拉·莱特（ペトラ・レイテ）**<br/>罗兹瓦尔领地中阿拉姆村的村民。<br/>茶色的头发上别着大大的蝴蝶结，梦想长大后到都市做衣服。<br/>因为记忆力和素质好，后来成为了罗兹瓦尔宅邸的女仆。<br/>在魔兽骚乱事件中被昴所救，而对昴有了爱慕之情。 | <img width="150" src="/res/img/character/Petra.jpg" /> |
| **帕特拉修（パトラッシュ，[Patrasche]）**<br/>在作为人的代步工具的地龙中，是一头以优秀著称的戴安娜种地龙。<br/>本来是在库鲁修宅邸受到重用的一头地龙，借白鲸战的机会成为了昴的爱龙。<br/>即使语言不通，她也能理解人类说的话，是非常聪明的地龙。<br/>对刚相遇不久的昴非常忠诚，与其说是忠诚不如说是保护欲很强，几次将昴从危机中救出。<br/>此外，她是雌性，奥托说她曾经说过“眼睛不能离开昴呢”。 | <img width="150" src="/res/img/character/Patrasche.jpg" /> |
| **法兰黛莉卡·鲍曼（フレデリカ・バウマン，[Frederica Baumann]）**<br/>作为女性来说，是一位身高很高、拥有澄澈翠瞳的美女——如果她把獠牙隐藏起来的话。<br/>曾在罗兹瓦尔宅邸工作过的女仆，最近复职了。<br/>体内流有亚人的血液，尖锐的牙齿即是受此影响。由于自卑，在笑的时候会用手遮住嘴。<br/>她作为拉姆和雷姆的前辈，对罗兹瓦尔的忠诚心和她的能力之高毋庸置疑。<br/>疼爱年下的孩子是她的兴趣，最近执着于教育新人女仆佩特拉。  | <img width="150" src="/res/img/character/Frederica.jpg" /> |
| **嘉飞尔·汀泽尔（暂译，ガーフィール・ティンゼル）**<br/>眼神凶恶，牙齿尖锐，如外表一样是位性急野蛮的人。<br/>在梅札斯领地中的秘密处所、圣域中生活的人们中的一员。<br/>非常有特点的说话方式和经常用的谜之惯用语常常搞得昴不知其所云。<br/>和罗兹瓦尔等人认识的时间很长，积极追求着拉姆。<br/>他的实力与他的性格相称，能轻松地将数百公斤的龙车猛扔出去。<br/>持有“地灵的加护”，只要站在地上就能恢复伤势。一度给昴制造了很多麻烦，和解后称昴为“大将”。 | <img width="150" src="/res/img/character/Garfield.jpg" /> |
| **莉莎·汀泽尔（暂译）**<br/>弗雷德莉卡和加菲尔姐弟的母亲。<br/>一开始是作为奴隶商人负债的抵押而准备被卖掉，途中遇到亚人族盗贼团<br/>（盗贼团其中一名成员即为弗雷德莉卡的父亲）,但弗雷德莉卡的父亲不久死去。<br/>莉莎带着仍是婴儿的弗雷德莉卡漂泊时被另一个亚人族集团抓住并因此邂逅卡菲尔的父亲，<br/>但由于种种原因不能和加菲尔的父亲待在一起的母子三人不得不再次漂泊，<br/>就在莉莎等三人走投无路之时梅扎斯家收留了他们。<br/>莉莎将弗雷德莉卡和加菲尔托付给罗兹瓦尔之后离开宅邸去寻找加菲尔的父亲，途中遭遇不测之后杳无音信。<br/>再次被众人找到时疑似失忆，被水门都市的一位好心的官员收留。 | <img width="150" src="/res/img/character/Lisa.jpg" /> |
| **琉兹·毕尔玛（リューズ・ビルマ）**<br/>外表年龄11、2岁，但是态度却很老成。<br/>圣域的代表人物，与加菲尔在一起经过了很长的一段时间。<br/>经常装作上了年纪的样子让周围的人非常困扰，实际上，她确实是圣域中资历最老的一位。<br/>虽然接受罗兹瓦尔的庇护在圣域生活，但是好像并不只是如此单纯的关系。<br/>长耳、长寿、美貌等条件都凑齐了，但是她却没有坦白过她的身世。 | <img width="150" src="/res/img/character/Luz.jpg" /> |



## 菲鲁特阵营

| 角色介绍 | 人物形象 |
|:---|:---:|
| **菲鲁特（フェルト，[Felt]）**<br/>15岁的流浪少女，有着枯黄的金发与好胜的红眸。<br/>自小在贫民窟长大，被世界眷顾而得到的风之加护，拥有超常的敏捷速度，为了自己的理想才逼不得已当上盗贼。<br/>真实身份为过去被掳走的先王的王弟——佛鲁德大人的千金，在莱因哈鲁特的拥护下成为了第5位国王候补人。 | <img width="150" src="/res/img/character/Felt.jpg" /> |
| **罗姆爷（ロム爷）**<br/>全名是克罗姆威尔，肌肉发达、身高超过2米的巨人族老人。<br/>居住在王都的贫民街，管理盗贼们藏匿及销售赃物的地方。<br/>与可憎的外表相反，意外地是个好心人，并将菲鲁特视为自己的家人一般照顾着她。 | <img width="150" src="/res/img/character/Rohm.jpg" /> |
| **莱因哈鲁特·范·阿斯特雷亚（ラインハルト・ヴァン・アストレア，[Reinhard Van Astrea]）**<br/>火红发色与清澈蓝眸的俊美青年，身材纤细而修长，当代剑圣，亦被称作“骑士中的骑士“。<br/>拥有剑圣血统及40种以上的加护，亦为历代最强剑圣，<br/>实力号称“地上最强”。是王都中无人不晓的名人，隶属于王城近卫队。<br/>即使是假日，也会为帮助民众们而竭尽全力，是帮助昴和爱蜜莉雅摆脱猎肠者追杀的关键人物。 | <img width="150" src="/res/img/character/Reinhard.jpg" /> |
| **阿顿（又译：桶）**<br/>昴所命名的地痞三人组，顿珍汉的“顿”担当。<br/>三人中性格相对豁达开朗，有时也站在抑制爱打架的其他两人的立场上。<br/>实际擅长料理。 | <img width="150" src="/res/img/character/Dun.jpg" /> |
| **阿珍（又译：庆）**<br/>昴所命名的地痞三人组，顿珍汉的“珍”担当。<br/>三人中最容易生气且不明事理的类型，经常携带着小刀。<br/>虽然经常用刀威胁别人却不会捅人。 | <img width="150" src="/res/img/character/Zhen.jpg" /> |
| **阿汉（又译：康）**<br/>昴所命名的地痞三人组，顿珍汉的“汉”担当。<br/>三人中最为血气方刚，无论是怎样的对手都毫不留情地咬住不放。<br/>经常有在咬住后后悔的情况，但却毫不反省。 | <img width="150" src="/res/img/character/Han.jpg" /> |
