――交涉成立，和拿着手机的罗姆爷做了约定。只是――

“我能做的只是为你和菲尔特准备一交涉的地方。这之后，有关用魔法器和徽章交换的事情就要靠你自己交涉了哦”

“这是预先知道是赃物的交涉哦？罗姆爷也能够保障这东西的价值吧。那就很轻松了啦”

看着轻松快乐的昴，罗姆爷像是有什么要说一样皱起眉头。

不要大意啦，不要小看交涉啦，大概就是想说这类忠告吧。

但是，也许是觉得对喝干了酒脸开始发红的昴说也没有用吧――

“嘛。实际上就是这样。如果遇到更加恶劣的家伙的话，也许会看准了弱点加价……菲尔特倒不是这种扭曲的家伙”

“吃了别的晚饭，别随便的说这种话啊。你这家伙，逐渐变得不客气起来了啊”

粗糙的碟子里盛放着一些糊状的豆子样的东西，昴一边嚼着这种东西一边抱怨着。

最初对酒的厌恶感已经不知道飞到哪里去了，现在罗姆爷正往那已经空了的玻璃杯里倒酒，用那种豆子当下酒菜一点一点地喝着酒。

“话说，那么大的身体吃这么一点就够了吗？光靠吃空气是活不了的，难道是说只要有酒不吃饭也没问题的那种？”

“不知道巨族在燃料费上的优点吗。和战斗力相反饭量很少。因为这在战乱的时候被用在各种地方了”（※虽然是魔法世界，可这还是太不科学了吧）

对于昴朴素的疑问，罗姆爷的脸色变得严肃。

他把酒瓶直接对准嘴巴直接把酒灌了下去。

“因此口减少了很多。就连王都也是，没有见过其他巨族呐”

“不用吃饭却很强吗，好厉害啊。@#￥%……想吐”

“你这家伙，我在忧郁地说话的时候不带用这种态度的吧！”

炫耀自己的不幸的话听了只会然不爽。

对塞起耳朵将对话打断的昴，罗姆爷也断了让他听的念头开始吃起豆子。

两就着这难吃的豆子，无言地喝着酒度过了接下来的时光。

响起独特的敲门声时，已经是日光相当倾斜的时分了。

开始昏昏欲睡的昴抬起了头，对声音起反应的罗姆爷移动着巨大的身体灵巧地朝着门的方向走去。

与罗姆爷的巨大身躯相比，门显得相当的小。将耳朵贴到门上，罗姆爷带着一种奇妙的表情向门对面叫道――

“毒”

“陷阱”

“吾等高贵的龙族对”

这大概就是所谓的暗号和暗语吧。带着满足的神情罗姆爷打开了锁。看着他的背影，昴内心暗想这些还真是些没有品的暗号啊。随后――

“――让你久等了啊，罗姆爷。遇到了意外烦的对手。花了好多时间”

一少女一边亲切地向罗姆爷炫耀着自己的战果，一边进入仓库走到他身边。

这是一有着半长金发的少女。兔子一般的红色瞳孔，嘴角露出了恶作剧般的虎牙。娇小的身体看起来行动敏捷，穿着一身破旧的衣服。

虽然当时只有匆匆一瞥，但毫无疑问这就是那少女。

没想到出现的是自己见过的少女，昴嗖地一下站了起来。

咔哒，注意到站起来还盯着自己看的昴发出的声音，少女的笑容消失，换上了一副怀疑的表情。

“啊？是谁。喂。罗姆爷。我说过了会有好东西要拿过来，叫你不要让其他进来了吧？”

“我知道你的心情，但是那小鬼找你――找菲尔特有事所以才呆在这里的。嘛，不是无关员”

对于罗姆爷的回答，菲尔特露出了一副愈发怀疑的表情。

少女用手挡着自己没有什么料的胸部，大概偷到的战利品就藏在那里吧。

承受着昴无言的视线，冷静不下来的菲尔特转过头看着罗姆爷――

“啥啊，那小哥。――难道，不会是把我给卖了吧”

“我才不会做那种不仗义的事情呢。而且我觉得对你没有坏处哦”

单眼朝这边眨了一下，罗姆爷“喏”地征求着这边的同意。

体验着单眼眨眼的老头带给自己的恶心感觉，昴轻轻地点了点头。

对着仍然没有打消警戒的菲尔特，昴斟酌着用词――

“总之，先冷静下来。我对你没有恶意……呕呕呕呕呕呕呕呕呕呕”

――斟酌的语句，随着败给呕吐感而涌上来的东西一起吐了出来。

“呀呀呀呀呀呀呀呀呀呀呀呀呀呀――！！”

理所当然地对面两对昴的丑态发出了悲鸣。

昴双膝跪地，忍受着胃部绞痛的苦楚就这样把肚子里的东西全部吐了出来。

琥珀色的酒和黏糊糊的豆子混合在一起，做出了一副最恶心的作品。

“说什么对我有好处啊！这不是最糟糕的情况吗！！”

“最糟糕的是我这边啊！被抢走吃掉的晚饭和酒啊！酒很贵的啊！很贵的啊！？”

“烦！别叫！头一跳一跳地在响！好恶心！真的好难受呕呕呕呕呕呕呕呕呕呕”

“呀呀呀呀呀呀呀呀呀呀呀呀呀呀――！！”

昴懦弱的精神再度屈服于涌上来的呕吐感，引来了两的悲鸣。

――在昴几乎将自己的胃都吐出来之后，直至打扫了自己的呕吐物，为房间换了气，被罗姆爷交代“禁止喝酒”，这阵骚动撕破了贫民窟的黑夜一直持续着。

――夜更深了。

“――那么，重新振作精神开始交涉吧！”

拍了一下手，像是要扫清现场尴尬的气氛一般，昴如此说道。

场所从盗品库柜台的一侧移动到了仓库的内侧一小小的桌子这里。柜台那里现在还留着骚动的残留（臭味），菲尔特坚决拒绝在那地方进行交涉。

同坐在桌子一旁，菲尔特大口喝着罗姆爷拿过来的牛奶。

从她的红色瞳孔中，警戒心仍然没有打消，现在也还带着不小的不信任和不快感。看样子是被刚才的事情给搅的。

“不要摆出那么恐怖的表情嘛，笑一笑笑一笑。可爱的脸都浪费了哟？”

“别在这套交情，小哥。而且对你讲的话，我只对能够变成钱的地方感兴趣。能不能别说废话赶紧说主题？”

对昴用手指着自己的脸颊拼命地做出的这番表演，菲尔特的反应相当冷淡。

来到这世界之后，昴的交流能力在各种地方都挥空了。即使是超越了世界，令昴处于孤立的『绝对孤单的力量』仍起着作用。

“明明一切都失去了，却唯独对她无法放手，吗。呼呼呼。这就是所谓的因果啊，不这么想吗？”

“喂，罗姆爷。这牛奶，掺水了吧。很难喝哦”

“怎么连你都这样，对别好心拿出来的东西一直嚷嚷着难吃难吃什么的……”

完全不理会沉浸在自己的世界之中的昴，菲尔特和罗姆爷在一旁欢乐地互动着。

昴轻咳了一声将两的视线集中到自己身上――

“那么，开始交涉。那，菲尔特。――你拿着徽章吧？”

“……啊啊，拿着”

昴试着单刀直入地发问，菲尔特坦率地做出了简短的回答。

少女将手伸入怀中，从那里拿出了一东西静静地放在桌子上。

――那正是昴一直寻求着的徽章。那是一仿照龙的形象设计出来的独具匠心的徽章。

大小大概是胸章大小。材质虽然无法判断，但是看起来用的是高价的金属。按照昴的知识，这是仿照飞龙的正面进行的设计，徽章的中央――描绘着一头嘴巴嵌着一块红色的宝石的龙。

徽章的中心，红色的宝石散发着淡淡的光芒像一盏灯一般，看着这景象昴不禁失声了。

低估了徽章的价值的罗姆爷也闭了嘴只能发出唔唔的声音在一旁露出为难的样子。

“那么――”

打破这阵沉默的，是握住徽章的菲尔特。

菲尔特用这句话确认了一下两都已经回过神来之后，捏起桌子上的徽章――

“这次轮到你让我看看你的牌了。这徽章这么不得了，而且我为了弄到它可是费了好一番功夫。能够拿出能够与之相抵的牌，让我们双方都高兴吗？”

“即使你这么坏笑着看着我，我能够出得了的牌也只有一张。因为我可是完美的身无分文哦！”

老样子大家只要一听到『身无分文』马上就露出了厌恶的脸啊，昴如此想着。

暂且将心中的情感放下，有如宣言的一般，昴打出了最强的牌。

看着咚的一声放到桌子上的手机，如昴预想的一般菲尔特露出困惑的表情。

对菲尔特和罗姆爷当初一样的反应感到相当满意，昴启动了手机的照相功能。

“菜月闪光！！”

“呜啊、好晃眼！”

一阵闪光和机械的快门声之后，画像到手了。

虽然菲尔特对昴没有礼貌的摄影一副想要抱怨的表情，但昴在她开口之前就将手机的画面推到了她眼前。

看着上面显示的自己的样子，少女红色的双眼睁得大大的。（※从前面就想吐槽了，中世纪的一辈子都不知都有没机会用一次镜子，鬼才知道上面拍的是自己啊，顶多觉得这张画很精美罢了）

“这是……”

“对，就是菲尔特！这是可以剪切时间保存为画面的魔法器。我能出的牌只有这一张。那徽章，我想要用这东西来交换”

第一手就出了最大的牌，靠着气势保持着优势推进着交涉。

昴采取的是交涉中的一种惯用的手法。视场合而定有可能直接决定交涉结果的强力的手段。

本来，这同样暗示这边除了那就没有其他牌了，昴实际上就是嘴上说说而已。

“原来如此，这还真是厉害啊。那，罗姆爷。这魔法器要是卖掉值多少？”

盯着画面，不住的点头的菲尔特之后的反应相当淡泊。

既没有双眼放光，也没有把手机拿到手里瞧。从效果来看，比起对魔法器本身，她仅仅是对这东西的高价值感兴趣。

“男的烂漫即使是在异世界也只有男能够理解吗！是这样吗！？”

“别在那叫啊，真难看。别动摇啊，你很有自信的吧。那么，那到底怎么样，罗姆爷”

该说是轻浮吗，还是该说奇妙的有一种男子气概的感觉呢，菲尔特神情自若毫无动摇。

对着见面的感觉比自己小两三岁的少女昴情不自禁地想要想她鞠躬叫“大姐”。

就在和男子气概还差得远的昴在心中如此腹诽的时候，菲尔特向站在旁边的罗姆爷询问手机的价值。

她打从心底对手机不感兴趣般的用手指摸着手机的侧面――

“要用魔法器和我换？虽然是没见过的东西，但要是比这嵌有宝石的徽章还值钱的话那就万万岁了。关于这一点，看起来并没有多可疑”

“嘛，如果要说具体数字的话实在无法想象。我也没有经手过魔法器呐。但是，虽然那徽章是相当有价值的东西，但这魔法器不会劣于它。――也就是说，这交涉对你相当有利。这就是我的结论”

“是吗是吗，那、这不是挺好的吗”

得到了罗姆爷的有力保证，菲尔特看起来挺高兴的。

虽然和预定稍稍有点不同，但是昴像是大体的目的已经达成般地充满喜悦。

可是，伸过手去的昴却被菲尔特一句“等等”给制止了。

“我们相互出牌是已经完了。可是，我的漫天要价可是还没有结束哦？”

“我还没坏到那种程度。老头子都这么说了呐。这东西的价值在徽章之上这一点我承认。――不过我倒认为你说已经没牌可出了这点是在说瞎话”

站起来的菲尔特居高临下地俯视着坐在椅子上的昴。

看着对方双瞳之中透出的嗜虐，昴就像是被看透了内心一般只能不停地冒冷汗。

作为交涉用的牌，效果最大的手机已经打出去了。

但是，昴手上还有几件这世界大概没有的财物。

钱包里的硬币和会员卡，只要交涉得好应该也能换点钱。实在不行的话运动衫和运动鞋，甚至是提供技术也能用来换钱。

手机说到底是最强的牌，还有几张其他的牌在。本来，那些是今后为了在这世界认真的生活下去作为最初的投资的为数不多的几张牌。

像坏一般，如此订正着的菲尔特带着一副坏到底的表情如此爽快地宣言。

对于菲尔特像是要征求同意一般的话，昴看了一眼罗姆爷后“唔唔”地做出了肯定。

感觉就像是疼爱孙子的笨蛋爷爷一样。

菲尔特兴高采烈地看着罗姆爷将牛奶潺潺注入杯子。呆然地看着这样的少女的昴，吐了口气――

“可是，一开始就把那对手叫到这里来，就是说老早就打着要就价钱做一番交涉的主意吗”

“没错，知道我花了多少工夫吗，为了偷这东西。而且和我这样柔弱的进行交易的话，对方要是直接动手抢怎么办。会这么想的吧？”

“柔弱、吗……”

光看菲尔特纤细的小子来下判断的话，这么形容绝对没有错。

可是只要一想到她的精力充沛和死皮赖脸的话，实在是抗拒用“柔弱”这词来形容她。

而且回想起来，她在偷徽章的过程中，曾经一度对正处于生死关头的昴置之不理。

精神方面哪里柔弱了啊，这样一想不禁火冒三丈。

“话说，你对我完全没有印象吗？”

“？在哪见过吗？可是，如果不是相当印象深刻的见面方式的话我是没空去记的哦。本来，小哥你看起来就相当朴素的嘛”

“只有茶色的头发和衣服看起来比较显眼”，菲尔特一边笑着一边说道。

态度里感觉不到是在说谎，自己的平庸外貌被否定同时也令昴愕然。

异世界的情感也许完全荒废了也说不定。强盗杀现场被这么就忘掉了就是证据。

另一方面，又有在什么好处都没有的情况下帮助昴的萨特拉，以及就算昴在这里吐了也没有把他赶出门的罗姆爷在，所以现在完全搞不明白了。

大概在异世界也要视而定吧。只看着坏的方面做出评价是不行的。

“嘛，就当做菲尔特记忆力很糟糕好了。那么，现在等着的那什么时候到？”

“真是令生气的说话方式。我和她说过日落时工作应该就能完成了，叫她日落后过来……现在太阳也已经下山了，应该快了”

“怎么还没来啊？”，仿佛以这句话为标志。

两声尖锐的敲门声传到了仓库里。

仓库里的三相视了一眼，罗姆爷对着菲尔特问道：

“怎么没对暗号？”

“啊，暗号没有告诉她。大概是我的客，让我看看”

菲尔特跳了起来，朝着入口走去。

对菲尔特随随便便就把这里告诉了别的行为，昴包含着“这样没问题吗？”这层意思拍了拍罗姆爷的肩膀。

“嘛，又不是不认识的家伙。和她交往的时间也不短了……当保镖这种事就做一做吧”

因为被依赖了而情不自禁地高兴着的老爷爷从仓库里面拿出了一根棍子。

棍子的长度大概有竹刀那么长，材质是木头。棍子的一头有一些星星点点的突刺，一看就是被打到就会造成致命伤的东西。（※大概就是狼牙棒）

“这是叫钉头棒还是啥的吧，果然就算是到了异世界，棍棒也还是标准装备啊……”

用RPG的标准，就是五十金的桧木棒。

从外表看来不会背叛菲尔特的期望，从侧面看着罗姆爷的装备，昴如此评价。这时，菲尔特带着一种异常亲切的笑容回来了。

“果然是我的客。这边，要坐吗？”

挥手要求暗处的昴退下，她亲切的对象是身后的。

那就是交涉的对手啊，这么紧张地看着对手的昴，稍稍有点惊讶。

因为菲尔特招手请入的物，是一外貌非常漂亮的女性。

女性相当高，大约和昴差不多高，年龄二十多岁前半左右。

容貌是眼角下垂的稳重的美，病态般雪白的肌肤在昏暗的仓库中也显得相当显眼。

黑色外套像羽织一般。前开的外套内侧紧贴着肌肤的同色装束引注目。虽然纤弱却显出一副好身材。

而且对方和昴一样有着一头在这世界相当少见的黑发。及腰的长发绑成一束，现在她正在用手摆弄着发端。

总觉得这是一位有着妖艳气场的成熟的大姐姐。

对昴来说，这是一向无缘，经验值也少得可怜的稀有的角色。

老实说，心脏跳得砰砰的。

基于精神方面的优先顺位，昴不由得起身将座位让给了菲尔特。

菲尔特靠在空出的位子上，她的左边是带着棍子的罗姆爷，右边是难隐紧张的站立着的昴。

这位女性受到了出乎意料小题大做的迎接，但是她并没在意只是轻轻地点了一下头――

“无关员倒是挺多的”

“要是赖账就麻烦了呐。这是像我这种弱小的的智慧。喂，昴拿喝的过来”

对挥着手任意驱使起来的菲尔特，无法反对只能听从。

挑了比较不那么脏的玻璃杯倒上牛奶拿到两面前。

对昴的服侍，女性说了声谢谢轻轻地行了一礼，并开始像是估价一般望着这边――

“那边的老我倒是能理解，这边的小哥呢？”

从站着的样子和气氛感觉到昴的怯场了吧。

对女性纯粹的疑问，菲尔特做出了一副马上进入主题一般的坏笑――

“这边的小哥是你的对手。我的另一交涉对手哦”

如此这般，宣告了一切的开始。
